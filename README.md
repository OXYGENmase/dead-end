# Dead End

Zombie tower defense. Build a maze. Kill zombies. Survive.

## Run

```bash
python main.py
```

Or on Windows:
```bash
run.bat
```

## Controls

| Key | Action |
|-----|--------|
| Click | Place tower |
| R-Click | Cancel |
| Space | Start wave |
| ESC | Pause |
| F3 | Debug overlay |
| F5 | Screenshot |
| F6 | Snapshot |

## Dev

```bash
python test.py      # Run validation tests
python debug.py     # Analyze snapshot files
```

## Requirements

- Python 3.11+
- Pygame-CE 2.5+

```bash
pip install -r requirements.txt
```
